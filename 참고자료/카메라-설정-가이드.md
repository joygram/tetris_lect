# 카메라 설정 가이드 (테트리스 프로젝트)

테트리스 튜토리얼에서 **Main Camera** 위치·확대를 바꿀 때 참고하는 문서입니다.  
5단계 [테스트 전 필수 설정](../단계별-학습/5단계-입력-처리.md#테스트-전-필수-설정-카메라게임-보드-표현)에서 **Position (5, 10, -10)**, **Orthographic**, **Size 10~12**를 안내합니다.

---

## 카메라가 이상할 때 (먼저 확인)

**증상**: 블록이 안 보인다 / 화면이 너무 멀다·가깝다 / 비스듬히 보인다 / 검은 화면이다.

**한 번에 맞추기** (Play **끄고** 적용):

1. **[Hierarchy > Main Camera]** 클릭
2. **[Inspector > Transform > Position]** → **X: 5**, **Y: 10**, **Z: -10** 입력 후 **Enter**
3. **[Inspector > Camera > Projection]** → **Orthographic** 선택
4. **[Inspector > Camera > Size]** → **10** 또는 **12** 입력 후 **Enter**
5. **Ctrl+S** 로 씬 저장

**확인**: **Game** 뷰 탭에서 보드(또는 블록)가 보이는지 봅니다. Scene 뷰가 아니라 **Game** 뷰입니다.

| 증상                | 확인할 것                                                                                                 |
| ------------------- | --------------------------------------------------------------------------------------------------------- |
| 블록/보드가 안 보임 | Position이 **(5, 10, -10)** 인지, **Size 10~12** 인지. Z가 **음수**(-10)여야 카메라가 보드 앞에 있습니다. |
| 화면이 비스듬함     | **Projection**이 **Orthographic**인지. Perspective면 원근 때문에 기울어 보입니다.                         |
| 수정해도 적용 안 됨 | **Play를 끈 상태**에서 수정했는지. 수정 후 **Ctrl+S**로 씬을 저장했는지.                                  |
| Game 뷰가 검은색    | **Hierarchy**에서 **Main Camera**가 있고, **Tag**가 **MainCamera**인지 확인.                              |

아래 **카메라가 고정되어 있는 이유**, **에디터에서 위치 변경이 안 될 때**도 참고하세요.

---

## 카메라가 고정되어 있는 이유

- 이 튜토리얼에는 **카메라를 움직이는 코드**가 없습니다. 씬에 한 번 배치한 뒤 그대로 두는 방식입니다.
- **유일한 기준**은 **씬 파일에 저장된 Main Camera 설정**입니다. 에디터에서 바꾼 값이 **씬에 저장되지 않으면** 다시 열 때 이전 상태로 돌아갑니다.
- **Play 모드 안에서** 바꾼 값은 **Play를 끄면 모두 되돌아갑니다.**

**조정 방법**: **Play를 끄고** → Hierarchy에서 **Main Camera** 선택 → Inspector에서 Position·Size 수정 → **Ctrl+S**로 씬 저장.

---

## 에디터에서 위치 변경이 안 될 때

| 확인 항목            | 설명                                                                                                |
| -------------------- | --------------------------------------------------------------------------------------------------- |
| **Game 뷰**          | Scene 뷰가 아니라 **Game** 뷰 탭에서 확인. 카메라 변경은 Game 뷰에 반영됩니다.                      |
| **Main Camera 선택** | Hierarchy에서 **Main Camera**를 선택한 뒤 수정. Tag가 "MainCamera"인 카메라만 Game 뷰에 사용됩니다. |
| **값 적용**          | Position·Size 숫자 입력 후 **Enter** 또는 다른 칸 클릭.                                             |
| **씬 저장**          | 수정 후 **Ctrl+S**. 씬 탭 옆 **\*** 가 있으면 저장되지 않은 변경입니다.                             |

그래도 안 되면: Position **(5, 10, -10)**, Size **10**을 **직접 입력** → Enter → **Ctrl+S**.

---

## 포지션 지정 후 편집 기능으로 수정이 안 될 때

| 확인 항목              | 설명                                                                                                                                       |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **Play 끄기**          | Play 중에 바꾼 값은 모두 되돌아갑니다.                                                                                                     |
| **Main Camera만 선택** | 이동 툴(**W**)로 드래그할 때 Hierarchy에서 **Main Camera**가 선택된 상태인지 확인. 다른 오브젝트가 선택돼 있으면 그쪽만 움직입니다.        |
| **Inspector 잠금**     | 자물쇠는 **기본값이 잠금 해제**. 잠근 경우에만 다른 오브젝트를 눌러도 잠긴 오브젝트가 계속 보입니다. 잠금 해제 후 Main Camera를 다시 선택. |
| **숫자 직접 입력**     | 드래그가 안 먹히면 **Transform > Position** X, Y, Z에 숫자 입력 후 **Enter** → **Ctrl+S**.                                                 |
