# 4단계 확장 실습

## 📌 이 실습들은?

4단계(블록 이동 구현)에서 쓰는 **Transform**(위치·크기·회전, 부모·자식)과 **변수·멤버·배열**을 **점증적으로** 연습할 수 있는 개별 실습입니다.

- **이전 단계(1~3단계)를 다 보지 않았어도** 각 파일만 따라 하면 진행할 수 있습니다.
- 각 실습 문서에 **이전 단계·개념 문서 링크**가 있으므로, 필요할 때만 참고하면 됩니다.
