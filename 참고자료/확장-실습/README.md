# 단계별 확장 실습

## 📌 이 실습들은?

**1~8단계에 걸쳐** 각 단계에서 쓰는 개념(변수, 배열, Transform, Input, UI 등)을 **전용 씬에서 점진적으로** 연습할 수 있는 **개별 파일**입니다.

- **이전 단계를 다 보지 않았어도** 각 실습 파일만 따라 하면 진행할 수 있습니다.
- 각 실습 문서에 **이전 단계·개념 문서 링크**가 있으므로, 필요할 때만 참고하면 됩니다.
