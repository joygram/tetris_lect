## 🎯 목표

- **2차원 배열** grid를 두고, 한 행의 모든 칸이 비어 있지 않은지 검사하는 **IsRowFull(row)** 함수를 만든다.
- Start()에서 몇 번째 행이 가득 찼는지 Debug.Log로 출력한다.

---

### 수행 과정

1. **빈 GameObject 생성**  
   - Hierarchy 우클릭 → Create Empty → 이름: **"Stage6ExpandTest"**

2. **C# 스크립트 생성**  
   - 이름: `Stage6ExpandTest`

3. **다음 코드 작성**
```csharp
using UnityEngine;

public class Stage6ExpandTest : MonoBehaviour
{
    private int width = 4;
    private int height = 4;
    private int[,] grid;  // 0=빈칸, 1=채워짐

    void Awake()
    {
        grid = new int[width, height];
        // 2번째 행(y=2)만 다 채우기
        for (int x = 0; x < width; x++)
            grid[x, 2] = 1;
    }

    void Start()
    {
        for (int y = 0; y < height; y++)
        {
            bool full = IsRowFull(y);
            Debug.Log($"행 {y} 가득 찼나? {full}");
        }
    }

    bool IsRowFull(int row)
    {
        for (int x = 0; x < width; x++)
        {
            if (grid[x, row] == 0)
                return false;
        }
        return true;
    }
}
```

4. **스크립트를 Stage6ExpandTest에 추가 후 Play**  
   - Console: 행 0·1·3은 false, 행 2만 true 로 나오는지 확인

5. **직접 해보기**: 행 0도 다 채운 뒤 `grid[x, 0] = 1;` (Awake에서) 추가하고, 행 0도 true가 나오는지 확인해 보세요.
