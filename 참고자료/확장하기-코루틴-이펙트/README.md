# 확장하기: 코루틴으로 이펙트 붙이기

## 📌 이 강좌는?

**8단계까지 완료한 뒤** "조금 더 재미있게 만들고 싶다"할 때 보는 **선택 강좌**입니다.

**코루틴**을 이용해 다음 두 가지 이펙트를 단계별로 붙여 봅니다.

- **게임 시작 시**: 화면에 **3, 2, 1, Go!** 카운트다운을 보여 준 뒤 첫 블록이 나오게 하기
- **줄이 사라질 때**: 줄이 **잠깐 깜빡이거나** 대기한 뒤 사라지게 하기

**코루틴이나 이펙트가 어렵게 느껴져도 괜찮습니다.**  
각 문서에는 **"처음 보시는 분께"**, **"한 줄 요약"**, **"막혔을 때 확인할 것"** 같은 안내를 넣어 두었습니다. **코드를 그대로 복사해서 붙이고** 동작만 먼저 확인한 뒤, "왜 이렇게 하면 되는지"는 천천히 읽어 보시면 됩니다.
