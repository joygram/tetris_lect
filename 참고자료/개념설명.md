# 테트리스 게임 만들기 - 개념 설명서

## 📖 이 문서는?

이 문서는 **기초 개념 학습**을 위한 목차입니다.  
단계별 실습에서 **복기**한 뒤, 아래 표에서 항목을 클릭하면 **해당 개념 문서**로 이동하며, **점진적으로 확장된 실습**으로 이어집니다.

**중학생도 이해할 수 있도록** 쉬운 예시와 **수행 과정(실습)**을 포함했습니다.

**🔤 영어 표기 읽기가 어렵다면** → **[영어 용어 발음·표기](./영어-용어-발음-표기.md)** (한글 발음 + 발음기호) 참고.

**🛠️ Unity에서 개념 실습하기**  
개념 실습을 **테트리스 게임과 분리된 씬**에서 단계적으로 하려면, 먼저 **[개념 실습 테스트 환경 가이드](./개념-실습-테스트-환경-가이드.md)**를 읽고 전용 씬을 만들어 두세요. 프로젝트와 상관없이 씬에서만 테스트할 수 있습니다.

## 💡 학습 팁

1. **하나씩 이해하기**: 모든 개념을 한 번에 이해하려 하지 말고, 하나씩 차근차근 배우세요.
2. **실습 먼저 하기**: 각 개념 문서의 **🛠️ 실습**을 따라 해보면 이해가 빠릅니다.
3. **오류는 친구**: 오류가 나면 왜 났는지 생각해 보세요.
4. **주석 활용**: 코드에 주석을 달아 두면 나중에 다시 볼 때 쉽습니다.
5. **작은 것부터**: 큰 기능을 한 번에 만들지 말고, 작은 기능부터 하나씩 완성하세요.
