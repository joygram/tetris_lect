# 반복문 (Loop)

## 📖 개념 설명

**반복문**은 같은 작업을 여러 번 반복합니다.

#### for 반복문
```csharp
// 0부터 9까지 출력
for (int i = 0; i < 10; i++)
{
    Debug.Log(i);
}
```

#### foreach 반복문
```csharp
// 리스트의 모든 항목을 확인
foreach (var cell in cells)
{
    Debug.Log(cell);
}
```

**테트리스 예시**:
```csharp
// 모든 행을 확인하여 가득 찬 행 찾기
for (int y = 0; y < height; y++)
{
    if (HasLine(y))
    {
        DeleteLine(y);
        RowDown(y);
    }
}
```

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `LoopTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;
using System.Collections.Generic;

public class LoopTest : MonoBehaviour
{
    void Start()
    {
        // for: 0부터 4까지 출력
        Debug.Log("=== for 반복문 ===");
        for (int i = 0; i < 5; i++)
        {
            Debug.Log("i = " + i);
        }
        
        // foreach: 리스트 순회
        Debug.Log("=== foreach 반복문 ===");
        List<string> names = new List<string>();
        names.Add("사과");
        names.Add("바나나");
        names.Add("오렌지");
        
        foreach (var name in names)
        {
            Debug.Log("과일: " + name);
        }
    }
}
```

3. **Play 후 Console 확인**
   - "i = 0" ~ "i = 4" 출력
   - "과일: 사과", "과일: 바나나", "과일: 오렌지" 출력

4. **직접 해보기**: for 문을 `i < 10`으로 바꿔서 0~9까지 출력해 보세요.
