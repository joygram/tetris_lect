# 클래스 (Class)

## 💡 초심자를 위한 한 줄 요약

**클래스** = **저장소(변수)** 와 **그 저장소를 다루는 함수**를 한 덩어리로 묶어 둔 것입니다.  
"왜 클래스에 넣나요?" → **관련된 저장소와, 그걸 다루는 일을 한 곳에 모아 두기 위해서**입니다.

**용어 정리** (클래스 안에 넣었을 때):
- **멤버** = 클래스 안의 **변수** (저장소)
- **메소드** = 클래스 안의 **함수** (보통 그 저장소를 읽거나 바꾸는 역할)

더 자세한 설명은 **[멤버와 메소드](./멤버와메소드.md)** 문서를 먼저 읽어 보세요.  
"저장소를 다루는 함수"가 뭔지, 왜 클래스에 넣는지 단계별로 설명되어 있습니다.

---

## 📖 개념 설명

**클래스**는 관련된 변수(멤버)와 함수(메소드)를 묶은 설계도입니다.

```csharp
public class Car
{
    // 멤버 = 클래스 안의 저장소(변수)
    public string color;
    public int speed;
    
    // 메소드 = 클래스 안의 함수 (저장소를 이용한 일)
    public void Drive()
    {
        Debug.Log("달려요!");
    }
}

Car myCar = new Car();
myCar.color = "빨강";
myCar.Drive();
```

**테트리스 예시**:
```csharp
public class TetrisBlock : MonoBehaviour
{
    private Vector2Int position;
    private List<Vector2Int> cells;
    
    public void MoveLeft() { ... }
    public void Rotate() { ... }
}
```

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `ClassTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;

public class ClassTest : MonoBehaviour
{
    void Start()
    {
        // Player 객체 생성
        Player p = new Player();
        p.name = "철수";
        p.score = 0;
        
        Debug.Log(p.name + "의 점수: " + p.score);
        p.AddScore(100);
        Debug.Log("점수 추가 후: " + p.score);
    }
}

// 별도 클래스 정의 (같은 파일에)
public class Player
{
    public string name;
    public int score;
    
    public void AddScore(int points)
    {
        score += points;
    }
}
```

3. **Play 후 Console 확인**
   - "철수의 점수: 0", "점수 추가 후: 100" 출력

4. **직접 해보기**: Player 클래스에 `int level` 변수(멤버)와 `LevelUp()` 함수(메소드)를 추가해 보세요.

5. **이해해 보기**: `name`, `score`는 **멤버**(저장소), `AddScore`는 **메소드**(그 저장소를 다루는 함수)입니다. [멤버와 메소드](./멤버와메소드.md) 문서에서 왜 이렇게 묶는지 더 자세히 볼 수 있습니다.
