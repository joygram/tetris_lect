# 조건문 (If-Else)

## 📖 개념 설명

**조건문**은 "만약 ~라면"을 표현합니다.

```csharp
int age = 15;

if (age >= 18)
{
    Debug.Log("성인입니다");
}
else
{
    Debug.Log("미성년자입니다");
}
```

**테트리스 예시**:
```csharp
// 블록이 이동 가능한지 확인
if (ValidMove(new Vector2Int(-1, 0)))
{
    position.x--;  // 이동 가능하면 왼쪽으로 이동
}
```

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `ConditionTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;

public class ConditionTest : MonoBehaviour
{
    void Start()
    {
        int score = 150;
        
        if (score >= 200)
        {
            Debug.Log("우수! 200점 이상");
        }
        else if (score >= 100)
        {
            Debug.Log("좋아요! 100점 이상");
        }
        else
        {
            Debug.Log("다시 도전해 보세요!");
        }
        
        // 테스트: score 값을 50, 120, 250으로 바꿔서 실행해 보세요
    }
}
```

3. **Play 후 Console 확인**
   - score가 150이므로 "좋아요! 100점 이상" 출력

4. **직접 해보기**: `score` 값을 50, 120, 250으로 바꾼 뒤 각각 어떤 메시지가 나오는지 확인해 보세요.
