# 이벤트 (Event)

## 📖 개념 설명

**이벤트**는 특정 상황이 일어났을 때 반응하는 일입니다.

**테트리스 예시**:
- 블록이 바닥에 닿으면 → 행 제거 확인
- 행이 가득 차면 → 점수 추가
- 10줄 제거하면 → 레벨 증가

---

## 🛠️ 실습: "조건이 만족되면" 동작하기

**목표**: 점수가 100을 넘으면 이벤트(로그 출력)가 발생하도록 합니다.

### 이 실습 전용 씬 만들기 (환경 설정)

**먼저 이 실습 전용 씬을 만든 뒤** 아래 순서로 진행합니다. (자세한 씬 만드는 방법은 [개념 실습 테스트 환경 가이드](../개념-실습-테스트-환경-가이드.md)의 **씬 만드는 방법** 참고.)

**이 실습만** 할 전용 씬이 없으면 아래 순서로 **한 번만** 만듭니다. (이미 `ConceptTest_Event` 씬이 있으면 **File → Open Scene**으로 열고 아래 실습만 이어서 하세요.)

1. **File** → **New Scene** → **Basic (Built-in)** 또는 **Empty** → Create
2. **File** → **Save As** → 저장 위치: **Assets/Scenes** → 파일 이름: **ConceptTest_Event** → Save
3. 아래 실습은 **모두 이 씬에서** 진행합니다. (다른 문서로 왔다 갔다 하지 않습니다.)

---

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `EventTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;

public class EventTest : MonoBehaviour
{
    private int score = 0;
    private bool alreadyCelebrated = false;  // 한 번만 축하하기 위해
    
    void Update()
    {
        // 매 프레임 점수 증가 시뮬레이션 (실제로는 줄 제거 시 증가)
        score += 1;
        
        // 이벤트: 점수가 100을 넘으면 한 번만 "100점 달성!" 출력
        if (score >= 100 && !alreadyCelebrated)
        {
            Debug.Log("이벤트: 100점 달성!");
            alreadyCelebrated = true;
        }
    }
}
```

3. **빈 GameObject에 스크립트 추가 후 Play**
   - Console에서 score가 100이 되는 순간 "이벤트: 100점 달성!" 한 번 출력되는 것 확인

4. **정리**: "블록이 바닥에 닿으면", "행이 가득 차면" 같은 **상황**이 일어날 때 코드가 반응하는 것이 이벤트입니다.
