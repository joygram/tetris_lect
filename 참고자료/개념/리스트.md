# 리스트 (List)

## 📖 개념 설명

**리스트**는 배열과 비슷하지만 크기가 자유롭게 변할 수 있습니다.

```csharp
// 리스트 생성
List<int> numbers = new List<int>();

// 항목 추가
numbers.Add(10);
numbers.Add(20);
numbers.Add(30);

// 항목 개수
int count = numbers.Count;  // 3
```

**테트리스 예시**:
```csharp
List<Vector2Int> cells = new List<Vector2Int>();
cells.Add(new Vector2Int(0, 0));
cells.Add(new Vector2Int(1, 0));
```

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `ListTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;
using System.Collections.Generic;

public class ListTest : MonoBehaviour
{
    void Start()
    {
        List<int> scores = new List<int>();
        scores.Add(100);
        scores.Add(200);
        scores.Add(300);
        
        Debug.Log("개수: " + scores.Count);
        Debug.Log("첫 번째: " + scores[0]);
        
        foreach (int s in scores)
        {
            Debug.Log("점수: " + s);
        }
    }
}
```

3. **Play 후 Console 확인**
   - 개수 3, 첫 번째 100, 점수 100/200/300 출력

4. **직접 해보기**: `scores.Add(400);` 을 추가한 뒤 Count와 출력 결과를 확인해 보세요.
