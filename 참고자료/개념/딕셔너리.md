# 딕셔너리 (Dictionary)

## 📖 개념 설명

**딕셔너리**는 키(Key)와 값(Value)을 쌍으로 저장합니다. 사전처럼 찾아볼 수 있습니다.

```csharp
Dictionary<string, int> scores = new Dictionary<string, int>();

scores["홍길동"] = 1000;
scores["김철수"] = 2000;

int hongScore = scores["홍길동"];  // 1000
```

**테트리스 예시**:
```csharp
Dictionary<int, Vector2Int[]> shapes = new Dictionary<int, Vector2Int[]>();
shapes[0] = new Vector2Int[] {...};  // I 모양
shapes[1] = new Vector2Int[] {...};  // O 모양
```

### 수행 과정

1. **Unity에서 새 C# 스크립트 생성**
   - 이름: `DictionaryTest`

2. **다음 코드 작성**
```csharp
using UnityEngine;
using System.Collections.Generic;

public class DictionaryTest : MonoBehaviour
{
    void Start()
    {
        Dictionary<string, int> scores = new Dictionary<string, int>();
        scores["철수"] = 100;
        scores["영희"] = 200;
        scores["민수"] = 150;
        
        Debug.Log("철수 점수: " + scores["철수"]);
        Debug.Log("영희 점수: " + scores["영희"]);
        
        // 키가 있는지 확인
        if (scores.ContainsKey("민수"))
        {
            Debug.Log("민수 점수: " + scores["민수"]);
        }
    }
}
```

3. **Play 후 Console 확인**
   - 철수 100, 영희 200, 민수 150 출력

4. **직접 해보기**: `scores["새이름"] = 300;` 을 추가하고 출력해 보세요.
