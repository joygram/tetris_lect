# 영어 용어 발음·표기 가이드

영어 표기가 읽기 어려울 때 **한글 발음(읽는 법)** 과 **의미**를 참고하세요.

**💡 읽는 법**  
- **한글 발음** 열만 보시면 됩니다. 그대로 읽으면 됩니다.  
- **발음기호(IPA)** 가 읽기 어려우시면 무시하셔도 됩니다. 한글 발음이 더 중요합니다.

## 💡 읽는 팁

- **한글 발음(읽는 법)** 만 보셔도 됩니다. 그대로 읽으면 됩니다.
- Unity 메뉴는 **Window → Layouts → Default**처럼 영어로 나오므로, 위 표에서 해당 단어를 찾아 읽어 보세요.
- 새로 나온 단어가 있으면 표에 추가해 두면 읽기 편합니다.

**이 문서는 [개념 설명](./개념설명.md) 및 단계별 학습(1~8단계)에서 나오는 영어 용어를 모은 것입니다. 전체 단어 발음은 이 문서 한 곳에서 찾을 수 있습니다.**
