# 8단계 이후 코드 리뷰 (목차)

8단계를 마친 뒤 **최종 상태의 스크립트**를 **리뷰 1단계 ~ 5단계**로 나누어 코드와 해설을 정리한 문서입니다.  
실습은 [8단계 - 게임 완성](../../단계별-학습/8단계-게임-완성.md)에서 하고, 여기서는 **전체 코드 이해·복습**용입니다.

**진행 순서**: 8단계 완성 → **코드 리뷰 1단계** → 2단계 → 3단계 → 4단계 → 5단계 (각 문서 하단 "다음"으로 이동).

## 리뷰 단계별 목차

| 리뷰 단계 | 스크립트 | 설명 |
| --------- | -------- | ----- |
| 1단계 | [GameBoard](./GameBoard.md) | 게임 보드 그리드, 블록 고정·행 제거·게임 오버 판단 |
| 2단계 | [GameController](./GameController.md) | 점수·줄·레벨, UI 텍스트, 게임 오버 표시 |
| 3단계 | [InputHandler](./InputHandler.md) | 키 입력 → TetrisBlock 이동·회전·하드 드롭 호출 |
| 4단계 | [Spawner](./Spawner.md) | 다음 블록 랜덤 생성, 게임 오버 시 중단 |
| 5단계 | [TetrisBlock](./TetrisBlock.md) | 한 블록의 이동·회전·낙하·고정·ValidMove |
