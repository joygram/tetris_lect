# 8단계 이후 코드 리뷰 (목차)

8단계를 마친 뒤 **최종 상태의 스크립트**를 **리뷰 1단계 ~ 5단계**로 나누어 코드와 해설을 정리한 문서입니다.  
실습은 [8단계 - 게임 완성](../../단계별-학습/8단계-게임-완성.md)에서 하고, 여기서는 **전체 코드 이해·복습**용입니다.

**진행 순서**: 8단계 완성 → **코드 리뷰 1단계** → 2단계 → 3단계 → 4단계 → 5단계 (각 문서 하단 "다음"으로 이동).

**📌 주석·복기 안내**
- 코드 블록의 주석은 **핵심만** 표기했습니다. `[우리 멤버]` / `[Unity API]` 같은 "내가 만드는 것 vs 엔진이 주는 것" 구분은 [2단계](../../단계별-학습/2단계-게임-보드-만들기.md)·[4단계](../../단계별-학습/4단계-블록-이동-구현.md)·[5단계](../../단계별-학습/5단계-입력-처리.md)·[7단계](../../단계별-학습/7단계-점수-레벨-시스템.md)·[8단계](../../단계별-학습/8단계-게임-완성.md) 학습 문서를 참고하세요.
- 각 리뷰 문서의 **관련 단계**와 **복습** 표는 **기존 단계별 학습에서 배운 내용**과 연결해 복기할 수 있도록 했습니다.
