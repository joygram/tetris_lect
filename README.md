# Unity 테트리스 프로젝트 - 중학생 강의용

Unity로 구현한 클래식 테트리스 게임입니다.  
이 프로젝트는 중학생을 대상으로 한 프로그래밍 강의 자료로 제작되었습니다.

## ⚠️ 이 저장소를 Unity에서 “프로젝트로 열기”가 안 돼요

이 폴더는 **강의 문서(마크다운)** 만 있는 저장소입니다. **Assets**, **ProjectSettings**, **Packages** 같은 Unity 프로젝트 폴더가 없어서, 이 경로를 Unity에서 “Open” 해도 **프로젝트로 인식되지 않습니다.**

**올바른 방법**  
1. **Unity Hub** → **New Project** → **2D** 템플릿으로 **새 Unity 프로젝트**를 만듭니다. (예: 프로젝트 이름 `Tetris`, 위치 `d:\workspace\TetrisUnity` 등 **이 폴더와 다른 경로**)  
2. 그 **새 프로젝트**를 Unity로 연 뒤, **이 저장소의 강의 문서**(1단계 ~ 8단계)를 보면서 **그 프로젝트 안에서** 씬·스크립트·프리팹을 만듭니다.  
3. 이 저장소는 **문서만** 참고용으로 두고, 실제 게임은 **새로 만든 Unity 프로젝트**에서 작업하면 됩니다.

→ 자세한 첫 단계는 **[1단계: Unity와 C# 기초](./단계별-학습/1단계-Unity-기초.md)** 의 “1. 프로젝트 생성”을 따라 하세요.

## 📚 GitHub Pages 블로그

이 강의 자료는 **GitHub Pages 블로그**로 게시되어 있습니다.

### 🌐 블로그 접속

GitHub Pages가 활성화되면 다음 URL에서 접속할 수 있습니다:
- `https://[사용자명].github.io/[저장소명]/`

**URL 확인 방법**:
1. GitHub 저장소의 **Settings** → **Pages**로 이동
2. "Your site is live at..." 메시지에서 URL 확인
3. 또는 직접 계산: `https://[GitHub사용자명].github.io/[저장소이름]/`

예시:
- 사용자명: `myusername`, 저장소명: `tetris`
- 블로그 URL: `https://myusername.github.io/tetris/`

## 🚀 GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 푸시
2. **Settings** → **Pages** → **Source**에서 **GitHub Actions** 선택
3. 자동으로 배포됩니다

자세한 내용은 [DEPLOYMENT.md](./DEPLOYMENT.md)를 참고하세요.

## 프로젝트 구조

```
Assets/
└── Scripts/
    ├── GameBoard.cs        # 게임 보드 관리 (그리드, 행 제거 등)
    ├── TetrisBlock.cs        # 테트리스 블록 조각 클래스 (이동, 회전, 낙하)
    ├── Spawner.cs          # 테트리스 블록 생성
    ├── GameController.cs   # 게임 로직 제어 (점수, 레벨)
    ├── InputHandler.cs     # 입력 처리
    └── Block.cs            # 블록 컴포넌트
```

## 주요 기능

- ✅ 7가지 테트리스 블록 모양 (I, O, T, S, Z, J, L)
- ✅ 좌우 이동 및 회전
- ✅ 자동 낙하
- ✅ 빠른 낙하 (Soft Drop)
- ✅ 하드 드롭 (Hard Drop)
- ✅ 행 제거 시스템
- ✅ 점수 및 레벨 시스템
- ✅ 게임 오버 감지
