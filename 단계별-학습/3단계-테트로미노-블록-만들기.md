# 3단계: 테트로미노 블록 만들기

## 🎯 이 단계에서 배울 것

- 7가지 테트로미노 모양 정의
- Tetromino 클래스 구현
- 블록의 시각적 표현
- 리스트와 딕셔너리 사용

## 📝 실습 내용

### 1. Tetromino 클래스 기본 구조

`Assets/Scripts/Tetromino.cs` 생성:

```csharp
using UnityEngine;
using System.Collections.Generic;

public class Tetromino : MonoBehaviour
{
    public List<Vector2Int> blocks;  // 블록 위치 리스트
    
    void Start()
    {
        blocks = new List<Vector2Int>();
        // I 모양 블록 정의
        blocks.Add(new Vector2Int(0, 0));
        blocks.Add(new Vector2Int(1, 0));
        blocks.Add(new Vector2Int(2, 0));
        blocks.Add(new Vector2Int(3, 0));
    }
}
```

### 2. 블록 시각화

각 블록 위치에 큐브 생성하여 표시

## ⚠️ 실제 구현과의 차이점

**블로그 설명**: 간단하게 이해하기 위해 (0, 0)부터 시작하는 상대 좌표로 설명합니다.

**실제 구현**:
- 블록의 **상대 좌표**는 (0, 0)부터 시작하는 것이 맞습니다 (블록 내부의 상대 위치)
- 하지만 블록의 **초기 위치**는 (0, 0)이 아닙니다:
  ```csharp
  // 실제 코드 (Tetromino.cs)
  position = new Vector2Int(gameBoard.width / 2 - 1, gameBoard.height - 1);
  // 즉, (4, 19) - 보드 중앙 상단에 배치됩니다
  ```
- 블록의 상대 좌표는 `shapes` 딕셔너리에 저장되며, 실제 위치는 `position` 변수에 저장됩니다
- 최종 Unity 월드 좌표는 `transform.position = new Vector3(position.x, position.y, 0)`로 변환됩니다

**중요**:
- 상대 좌표 (0, 0) = 블록의 기준점 (보통 왼쪽 아래 모서리)
- 실제 위치 = `position` 변수 값 (예: 4, 19)
- Unity 월드 좌표 = `transform.position` 값
