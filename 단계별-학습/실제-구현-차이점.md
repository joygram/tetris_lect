# 구현 방식 정리

이 강의에서 사용하는 테트리스 구현 방식을 요약한 문서입니다.  
실제 코드는 `Assets/Scripts/` 폴더의 스크립트를 참고하세요.

---

## 📍 좌표·위치

- **블록 초기 위치**: Spawner GameObject 위치에서 생성. 씬에서 Spawn Point 위치를 설정하면 그 좌표에 생성됩니다.
- **좌표 처리**: Unity 월드 좌표를 사용하며, `Vector3Int.RoundToInt()`로 정수 칸에 맞춥니다. 게임 보드의 논리적 좌표(0,0부터 width×height)와 Unity 월드 좌표는 씬 설정에 따라 연결됩니다.
- **GameBoard**: 논리적 좌표계는 (0, 0)부터 시작합니다. Unity 월드 좌표는 GameBoard 오브젝트 Transform은 씬에서 자유롭게 설정 가능합니다.

---

## 🎮 블록·이동

- **블록 생성**: Unity 에디터에서 만든 프리팹(I_Block, O_Block 등)을 Spawner가 Instantiate로 생성합니다. `transform.GetChild()`로 자식 큐브에 접근합니다.
- **이동**: `transform.position`을 직접 변경하고, `ValidMove()`로 범위·쌓인 블록 검사 후 이동합니다.
- **현재 블록**: InputHandler는 `FindObjectOfType<TetrisBlock>()`로 현재 조작 중인 블록을 찾아 MoveLeft, MoveDown 등을 호출합니다. 성능을 위해 한 번 찾아 변수에 저장해 재사용하는 방식도 가능합니다.

---

## ✅ 정리

| 항목        | 이 강의 방식                     |
| ----------- | -------------------------------- |
| 블록 생성   | 프리팹 기반, Instantiate         |
| 보드 데이터 | 2차원 배열 (grid)                |
| 좌표        | Unity 월드 좌표, RoundToInt 사용 |
| 이동        | transform.position 직접 변경     |
| 자식 접근   | transform.GetChild()             |
